# Hyprland - Fedora
Hyprland automatically built by GitHub Actions for Fedora

# Supported version
- Fedora 43
    - Hyprland v0.52.1

# Tips
```shell
# List all packages from a copr repo (e.g: solopasha:hyprland)
dnf list --available --repo=copr\* | grep "copr:copr.fedorainfracloud.org:solopasha:hyprland"
```
